Class Definition CC1G2BlockWriteOpSpecResult for LLRP parameter C1G2BlockWriteOpSpecResult.

Document Reference LLRP Specification Section 15.2.1.5.5.6

Document Reference LLRP Specification Section 16.3.1.5.5.6 

<p>Contains the result of a block write operation</p> 


<p>Readers that do not support C1G2 Block Write

SHALL set CanSupportBlockWrite to false in C1G2LLRPCapabilities. If such a Reader receives an ADD_ACCESSSPEC with an AccessSpec that contains this OpSpec parameter, the Reader SHALL return an error for that message and not add the AccessSpec.

void LLRP::CC1G2BlockWriteOpSpecResult::assimilateSubParameters ( CErrorDetails *  pError)
virtual

Assimilates the decoded sub-parameters into m_listAllSubParameters.

After fields are decoded, the CDecoder itself takes care of gathering the subparameters into m_listAllSubParameters. Once the end of the enclosing TLV (or message) is reached this assimilateSubParameters() function is called to create parameter refrences from the primary member variables.
