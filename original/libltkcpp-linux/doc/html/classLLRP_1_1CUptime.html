Class Definition CUptime for LLRP parameter Uptime.

Document Reference LLRP Specification Section 7.1.3.1.1.2

Document Reference LLRP Specification Section 16.2.2.2 

<p> The timestamps in LLRP messages or parameters can be either the uptime or the UTC time [UTC]. If a Reader has an UTC clock, all timestamps reported by the Reader

SHALL use an UTC timestamp parameter. If a Reader has no UTC clock capability, all timestamps reported by the Reader SHALL use the uptime parameter.

void LLRP::CUptime::assimilateSubParameters ( CErrorDetails *  pError)
virtual

Assimilates the decoded sub-parameters into m_listAllSubParameters.

After fields are decoded, the CDecoder itself takes care of gathering the subparameters into m_listAllSubParameters. Once the end of the enclosing TLV (or message) is reached this assimilateSubParameters() function is called to create parameter refrences from the primary member variables.
