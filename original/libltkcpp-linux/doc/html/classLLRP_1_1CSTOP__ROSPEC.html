Detailed Description

Class Definition CSTOP_ROSPEC for LLRP message STOP_ROSPEC.

Document Reference LLRP Specification Section 10.1.7

Document Reference LLRP Specification Section 16.1.9 

<p>This message is issued by the Client to the Reader. Upon receiving the message, the Reader stops the execution of the ROSpec corresponding to the ROSpecID passed in this message. STOP_ROSPEC overrides all other priorities and stops the execution. This basically moves the ROSpec's state to Inactive. This message does not the delete the ROSpec.</p> 


<p>ROSpecID: Zero is disallowed.</p>

void LLRP::CSTOP_ROSPEC::assimilateSubParameters ( CErrorDetails *  pError)
virtual

Assimilates the decoded sub-parameters into m_listAllSubParameters.

After fields are decoded, the CDecoder itself takes care of gathering the subparameters into m_listAllSubParameters. Once the end of the enclosing TLV (or message) is reached this assimilateSubParameters() function is called to create parameter refrences from the primary member variables.
