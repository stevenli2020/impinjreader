Detailed Description

Class Definition CImpinjRMCSentence for LLRP parameter ImpinjRMCSentence. 

<p>This custom parameter contains the current GPS information of the Reader's location as reported in NMEA RMC sentence format. If the GPS device has not acquired a location fix, the string is reported empty.</p>

void LLRP::CImpinjRMCSentence::assimilateSubParameters ( CErrorDetails *  pError)
virtual

Assimilates the decoded sub-parameters into m_listAllSubParameters.

After fields are decoded, the CDecoder itself takes care of gathering the subparameters into m_listAllSubParameters. Once the end of the enclosing TLV (or message) is reached this assimilateSubParameters() function is called to create parameter refrences from the primary member variables.
